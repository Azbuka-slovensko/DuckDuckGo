SeekingWhirl is a library for easy work with DuckDuckGo API

Get it!

Latest version: 1.0.0

Free and easy

Just add it to your project and use! You don't have to configure it or build - instalation is fully automatic.

In your dub.json:

{
	...
	"dependencies": {
		"seekingwhirl": "~>1.0.0"
	}
}

You can also visit page on code.dlang.org, or download zipped package on releases
